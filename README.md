# Analisis Data Tuberkulosis (Tuberculosis)

## Deskripsi Proyek

Proyek ini bertujuan untuk melakukan analisis data kejadian Tuberkulosis (Tuberculosis/TB) berdasarkan data yang diperoleh dari sumber terbuka. Analisis dilakukan untuk memahami pola persebaran kasus TB, tren kejadian dari waktu ke waktu, serta memberikan gambaran statistik deskriptif terhadap data yang digunakan.

## Dataset

Dataset yang digunakan berisi informasi mengenai insiden Tuberkulosis pada berbagai negara dan tahun pengamatan.

Variabel yang digunakan:

* **Entity** : Nama negara atau wilayah.
* **Code** : Kode negara.
* **Year** : Tahun pengamatan.
* **Incidence of Tuberculosis** : Jumlah insiden Tuberkulosis.

## Tujuan Analisis

1. Mengetahui karakteristik data Tuberkulosis.
2. Menghitung statistik deskriptif dataset.
3. Mengidentifikasi tren kejadian Tuberkulosis dari waktu ke waktu.
4. Membandingkan tingkat insiden Tuberkulosis antar negara atau wilayah.

## Metode Analisis

Analisis dilakukan menggunakan bahasa pemrograman Python dengan beberapa tahapan:

1. Import dataset.
2. Data cleaning dan pemeriksaan data.
3. Analisis statistik deskriptif.
4. Visualisasi data menggunakan grafik.
5. Interpretasi hasil analisis.

## Tools dan Library

* Python
* Pandas
* Matplotlib
* google collab

## Hasil yang Diharapkan

Hasil analisis diharapkan dapat memberikan gambaran mengenai kondisi Tuberkulosis secara global maupun regional serta membantu memahami pola perkembangan penyakit dari waktu ke waktu.
